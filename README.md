<h1>🌟 Portfolio Website Overview</h1> <br>
This is a modern and professional portfolio website designed to showcase Ayush Ghole’s skills, projects, and expertise. It provides a clean, interactive, and user-friendly experience for visitors.
<br>
<br>
<h1>✨ Key Features & Functionalities</h1> 
<b>🟢 1. Navbar (Sticky & Responsive)</b> <br>
<ul>
  <li>Contains navigation links: About, Projects, Contact, and Hire Me</li>
  <li>Mobile-friendly with a collapsible menu</li>
</ul>
<b>🚀 2. Hero Section (Introduction)</b> <br>
<ul>
  <li>Displays Ayush Ghole’s Name & Title with a typing animation effect</li> 
  <li>Includes a “Get Started” button that opens a modal contact form</li> 
  <li>Features a profile image with a shadow effect</li>
</ul>

<b>📂 3. Projects Section </b> <br>
<ul>
  <li>Project Name & Description</li>
  <li>Live Demo & GitHub Links</li> 
  <li>Embedded YouTube Video Previews
</li>
</ul>
<b>4. Contact Form (Modal Popup)</b> <br>
<ul>
  <li>Opens when clicking "Contact" (Navbar) or "Get Started" (Hero Section)</li> 
  <li>Contains fields: Name, Email, Contact, Address, and Message</li>
  <li>Includes validation for user inputs</li> 
  <li>Displays a beautiful success message using SweetAlert2</li>
</ul>

<b>🎨 5. UI/UX Enhancements </b><br>
<ul>
  <li>Glassmorphism effect on the modal</li>
  <li>Smooth transitions & animations for better engagement</li>
  <li>Fully responsive & mobile-friendly</li>
</ul>

<b>🛠️ Technologies Used </b> <br>
<ul>
  <li>React.js (Frontend framework)</li> 
  <li>Bootstrap (Responsive styling)</li> 
  <li>SweetAlert2 (Professional alerts & success messages)</li>
  <li>Typewriter Effect (Typing animation for Hero Section)</li>
</ul>

<b>🌟 User Experience Flow </b><br>
<ul>
  <li>User visits the website → Sees intro with a typing effect</li> 
  <li>Navigates through sections → Can explore projects & details</li>
 <li>Clicks "Get Started" button → Opens a modal form</li>
  <li>Fills out the contact form → Receives a success confirmation</li>
</ul>
